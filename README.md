# e-plantShopping

**e-plantShopping** is a modern web-based plant shopping application built using React and Redux.  
The application allows users to browse a variety of plants, add them to a shopping cart, and manage cart items efficiently.

## Project Overview

The **e-plantShopping** project is designed to simulate an online plant store where users can:
- View a list of available plants
- Add plants to a shopping cart
- Increase or decrease item quantities
- View the total number of items in the cart

This project focuses on implementing **Redux Toolkit** for global state management and follows best practices in React component design.
